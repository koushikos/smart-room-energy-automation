# Smart Room Energy Automation System

A beginner-friendly Smart Automation web application that monitors device usage and provides energy-saving suggestions.

## Features
- Device ON/OFF control
- Energy usage calculation
- Smart usage alerts
- Room selection
- Login/Register (localStorage)
- Dark/Light theme
- PDF energy report export
- Usage prediction logic

## Tech Used
- HTML
- CSS
- JavaScript
- localStorage
- Chart.js
- jsPDF

## Purpose
Built as an Ideathon project to demonstrate Smart Automation for energy efficiency in hostels and homes.

## How to Run
Open index.html using Live Server in VS Code.
